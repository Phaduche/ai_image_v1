# AI 생성 이미지 반복 노출 실험 웹사이트

## 연구 제목
AI 생성 이미지의 반복 노출이 현실 이미지 판별 능력에 미치는 영향: 단순노출효과를 중심으로

## 가설
AI 생성 이미지를 반복적으로 볼수록 실제 사진과 AI 생성 이미지를 구별하기 어려워진다.

## 웹사이트 기능
- 참가자 기본 정보 수집
- 이미지 반복 노출 단계
- 실제 사진 / AI 생성 이미지 판별 검사
- 친숙감, 자연스러움, 확신도, 진짜 같음, 신뢰도, 공유 의향 측정
- 반응 시간 자동 기록
- CSV 다운로드

## 이미지 넣는 방법
1. `images/` 폴더에 이미지를 넣습니다.
2. 예시 파일명:
   - `real_01.jpg`
   - `real_02.jpg`
   - `ai_01.jpg`
   - `ai_02.jpg`
3. `script.js`의 `imageSet` 배열에서 이미지 경로, 실제/AI 여부, 노출 횟수를 수정합니다.

```js
{ id: "ai_03", type: "ai", src: "images/ai_03.jpg", exposure: 5 }
```

- `type`: `"real"` 또는 `"ai"`
- `exposure`: 반복 노출 횟수. 예: 0, 2, 5, 10

## 실행 방법
그냥 `index.html`을 브라우저에서 열면 됩니다.

## 배포 방법
- Vercel, Netlify, GitHub Pages에 업로드하면 온라인 실험 링크로 사용할 수 있습니다.

## ISEF 연구 분석 아이디어
- 반복 노출 횟수별 진위 판단 정확도 비교
- AI 이미지와 실제 사진 간 정확도 변화 비교
- 친숙감과 진짜 같음 점수의 상관관계
- 친숙감이 반복 노출과 오판 사이를 매개하는지 분석
- AI 경험 수준이 결과를 조절하는지 분석
